核心视觉

页面不再以白色/暖白展开。开场是深暗的底色，一束从中心向上方渐亮的追光（warm spotlight），标题从这束光中浸出——像是黑暗舞台上，聚光灯亮起照亮主角的瞬间。

不是"加个动画"，而是整个页面的亮场过程：

  • 初始：深暗背景 + 中心微光呼吸
  • 阶段 1：光晕膨胀，暗角退散
  • 阶段 2：标题字符从光中浮现（每个字有微小时差）
  • 阶段 3：副标题、按钮、装饰金线陆续入场
  • 收束：光稳定为页面全局氛围，过渡到内容区

与之前方案的本质区别

之前（被拒）

机械黑边 → 不叙事、无情感、像视频播放器不是电影

现在

光影叙事 → 黑暗中的仪式感、期待、揭示。视觉上"大气"来自明暗对比和节奏控制，而非机械变化。

技术上

Hero 区域包装一个深色全屏容器，背景用多层 CSS radial-gradient 模拟聚光灯。
用现有的 data-section-theme 机制切换全局氛围——开场用 "dark"（暗场），滚动至内容区后过渡到现有的 "gold" 暖光。

标题逐字浮现用 CSS animation-delay 逐步 stagger，无需 JS。
