第 4 层：胶片质感 (Film Stock)

全局氛围层——不是"加滤镜"，而是像电影胶片一样有温度和触感

3 个全局氛围层

① 胶片颗粒

替换现有的稻纸纹理（SVG noise），换为更细腻、有方向性的胶片颗粒。颗粒密度在不同区域有微妙变化——暗区颗粒更明显，亮区几乎不可见。

用 CSS filter + 多层 SVG noise，opacity 约 0.02-0.04

② 暗角晕影

页面四角有极克制的暗角（vignette），不是 Instagram 滤镜那种强暗角，而是电影放映时画幅边缘的自然暗沉——几乎察觉不到，但它让视觉焦点始终保持在画面中央。

radial-gradient 叠加，透明度 max 0.15，边缘最暗

③ 暖调色彩偏移

页面整体偏暖金，但高光区微微偏冷（像 Kodak Portra 胶片的色彩特性）。不直接改色，而是在 body::before 的渐变中微调色温分布。

调整现有 ambient gradient 的色相分布

古典元素的去留

保留

  • 金色装饰线（ornamental-rule）→ 更细、更低调
  • 古典边角（classical-corners）→ 保留，仅 hover 可见
  • 墨晕光标 → 增强为追光

替换 / 移除

  • 稻纸纹理 → 胶片颗粒
  • 钻石装饰点（ornamental-diamond）→ 去掉，过于具象
  • 金色粒子（sparkle）→ 精简为仅在 Hero 的追光中出现
